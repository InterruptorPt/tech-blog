Lançámos o código de ["Web Monetization: Parte 2 - Verificar atividade"](/blog/web-monetization-part-two-checking-its-active) como um [pacote no NPM](https://www.npmjs.com/package/@interruptor/react-monetization). Neste momento, fornece duas coisas:

- `WebMonetizationProvider`: o componente que adiciona os ouvintes de eventos para acompanhar o estado atual de monetização.

```jsx
const MyApp = ({ children }) => {
  return <WebMonetizationProvider>{children}</WebMonetizationProvider>
}
```

- `useWebMonetization`: um _hook_ para ler o estado de Web Monetization dentro dos teus componentes.

```jsx
const {
  isActive,
  state, // "pending", "started" ou "stopped"
} = useWebMonetization()
```

Se quiseres dar uma olhada, podes encontrar o [pacote on npm](https://www.npmjs.com/package/@interruptor/react-monetization) e o [código no GitHub](https://github.com/InterruptorPt/react-monetization).
